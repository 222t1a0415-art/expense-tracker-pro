# 💰 Expense Tracker Pro

A modern and responsive Expense Tracker web application built using **HTML, CSS, JavaScript, Chart.js, and Local Storage**.

Track your income and expenses, visualize spending patterns, set budgets, generate reports, and export transaction data.

---

## 🚀 Features

### 📊 Dashboard

* Total Balance
* Total Income
* Total Expense
* Real-time updates

### ➕ Transaction Management

* Add Income and Expense transactions
* Select categories
* Add transaction dates
* Delete transactions
* Persistent storage using Local Storage

### 🔍 Search & Filters

* Search transactions by name
* Filter by date
* Filter by category

### 📈 Analytics

* Pie Chart for Expense Categories
* Bar Chart for Income vs Expense
* Monthly Expense Trend Chart

### 🎯 Budget Management

* Set monthly budget
* Budget alerts
* Remaining budget calculation

### 📄 Reports & Export

* Monthly financial report
* Export transactions as CSV

### 🌙 Dark Mode

* Light/Dark theme toggle
* Modern user interface

### 📱 Responsive Design

* Mobile-friendly layout
* Tablet support
* Desktop optimized

---

## 🛠️ Technologies Used

* HTML5

1. Clone the repository

2. Navigate to the project folder

```bash

```

3. Open `index.html` in your browser

No additional setup required.
